With this plugin you can automatically generate shipping labels for PostNord Skicka Direkt Business.  


== Installation ==

1. Upload `postnord-woocommerce.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Setup ==

1. Go to https://developer.postnord.com/ and sign up for a customer account.
2. Enter the API key into the plugin settings.
3. Enter the rest of the plugin settings. 


== Requirements  ==

WooCommerce >= v3.0
